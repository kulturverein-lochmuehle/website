## Resets

```html
<kvlm-typo>
  <h1>Heading 1</h1>
  <h2>Heading 2</h2>
  <h3>Heading 3</h3>
  <h4>Heading 4</h4>
  <h5>Heading 5</h5>
  <h6>Heading 6</h6>
  <p>
    Paragraph with some <strong>strong passages</strong>,
    but also <em>emphasized words</em>, or some <small>smaller text</small>.
  </p>
  <p>
    <mark>Marked text</mark> is also possible, as well as
    <del>deleted text</del> and <ins>inserted text</ins>.
  </p>
  <p>
    Some texts may contain <a href="#">links</a>, dates on <time datetime="2019-12-06">06.12.2023</time>, or complete addresses like <address>Street 1, 12345 City</address>.
  </p>
  <ul>
    <li>we may have</li>
    <li>unordered lists</li>
    <li>with some entries</li>
  </ul>
  <ol>
    <li>or ordered lists</li>
    <li>with some entries</li>
    <li>as well</li>
  </ol>
</kvlm-typo>
```
